HTML, or HyperText Markup Language, is the standard language used to create and structure web pages. It is not a programming language but a markup language that uses tags to define elements such as headings, paragraphs, images, links, tables, and multimedia. Every web page on the internet is built using HTML as its foundation, with tags that tell the browser how to display the content. A basic HTML document starts with a doctype declaration and contains two main sections: the head, which holds metadata, and the body, which contains the content visible to users. Modern HTML, known as HTML5, introduces semantic elements like header, footer, article, and section, along with support for audio, video, and canvas, making it powerful and versatile. HTML works hand in hand with CSS for styling and JavaScript for interactivity, together forming the core technologies behind the web.
