Formatting Tags

  • Bold

    • Renders text in a bold font.
    example: This is bold text.

  • Strong

    • (Semantic): Indicates that the enclosed text has strong importance and is typically rendered in bold.
    example: This is important information.

  • Italic:

  • (Physical): Renders text in an italic font.
    example:

    This is italic text.

  • (Semantic)

    • (Semantic): Indicates emphasized text and is typically rendered in italics.
    example:

    This is emphasized text.

  • Underline

    • (Physical): Underlines the enclosed text.
    example:

    This is underlined text.
